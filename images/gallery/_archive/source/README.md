# Gallery source material

These files are retained as source and provenance material for the Lipoderma gallery.
They are not loaded by the live site and are excluded from the GitHub Pages deployment artifact.

| Path | Original path | Category | Notes |
|---|---|---|---|
| `clinical-full/` | `images/gallery/_source/clinical-full/` | content source | Raw clinical image downloads used during gallery curation. |
| `visible-results/` | `images/gallery/_source/visible-results/` | content source | Extracted before/after source imagery. |
| `results-manifest.json` | `images/gallery/_source/results-manifest.json` | provenance JSON | Historical source mapping for the prior JPG gallery set. |

The live gallery runtime mapping remains `data/gallery.json`, which points to the active PNG case images.
